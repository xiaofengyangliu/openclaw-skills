# lunar-calendar - 农历公历互转工具

Openclaw Skills 集合中的农历公历互相转换工具。

## 功能特性

- ✅ 公历转农历
- ✅ 农历转公历
- ✅ 正确处理闰月
- ✅ 正确处理闰年 2月29日
- ✅ 支持多种日期输入格式
- ✅ 显示生肖和干支纪年
- ✅ 识别传统节日

## 安装

```bash
cd ~/.claude/skills/openclawSkills/lunar-calendar
npm install
```

或者在项目根目录执行：

```bash
npm run install-all
```

## 使用方法

```
/lunar-calendar <日期>
```

### 支持的输入格式

**公历:**
```
/lunar-calendar 2024-02-10
/lunar-calendar 2024/02/10
/lunar-calendar 2024年2月10日
```

**农历:**
```
/lunar-calendar 2024年正月初一
/lunar-calendar 2024年1月1日
/lunar-calendar 2023年闰二月初十
```

## 示例

### 示例 1: 公历转农历

```
/lunar-calendar 2024-02-10
```

输出:
```
📅 日期转换结果
────────────────
公历: 2024年2月10日
农历: 2024年正月初一日
────────────────
生肖: 龙
干支: 甲辰
节日: 春节
```

### 示例 2: 农历转公历（闰月）

```
/lunar-calendar 2023年闰二月初十
```

输出:
```
📅 日期转换结果
────────────────
农历: 2023年闰2月10日
公历: 2023年3月29日
────────────────
生肖: 兔
干支: 癸卯

ℹ️  注: 这是闰月日期
```

### 示例 3: 闰年 2月29日

```
/lunar-calendar 2024-02-29
```

输出:
```
📅 日期转换结果
────────────────
公历: 2024年2月29日
农历: 2024年2月20日
────────────────
生肖: 龙
干支: 甲辰

ℹ️  注: 这是闰年2月29日
```

## 支持范围

- 年份范围: **1900年 - 2100年**
- 自动验证日期合法性
- 正确处理 1900-2100 年间所有闰月
